# GazePilot /world assets

The rebuilt `/world` scene uses local runtime assets only. No external hotlinks are used.

## CC0 model assets

- Kenney City Kit Commercial, City Kit Roads, City Kit Suburban, Car Kit and Modular Buildings.
  - Source: https://kenney.nl/assets
  - License: Creative Commons CC0
  - Usage: district landmarks, side buildings, hangars, city massing and supporting street-kit assets.
- Quaternius Hoodie Character via Poly Pizza.
  - Source: https://poly.pizza/m/gKLBoRsyKe
  - License: CC0
  - Usage: active player avatar and NPC avatar variants with local material recoloring.

## Procedural scene assets

Sky, island terrain, roads, skyline silhouettes, mountains, lamps, trees, cargo props, gates, terminals, antenna rigs, runway details, drone/shuttle props and building overlays are generated from Three.js primitives inside the repository.

License: project-owned implementation assets, derived from code in `frontend-nextjs/components/world`.
