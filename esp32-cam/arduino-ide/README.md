# Arduino IDE Wrappers

Use estes sketches quando quiser gravar pelo Arduino IDE:

- `gazepilot_camera/gazepilot_camera.ino`: firmware completo de captura/envio de frames.
- `gazepilot_api_test/gazepilot_api_test.ino`: teste de conectividade (`/health`) e captura.

Ambos reaproveitam o código-fonte oficial em `esp32-cam/src/`.
